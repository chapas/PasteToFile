🎯 USAGE QUICK START
─────────────────────────────────────────────────────────────────────

1. INSTALL
   - Double-click: PasteToFile-Setup-1.0.0.exe
   - Follow wizard
   - Optional: Enable Windows startup
   - Complete installation

2. USE
   - Copy image/text to clipboard
   - Right-click in folder
   - Select: "Paste to file"
   - File created automatically

3. UNINSTALL
   - Control Panel > Programs > Uninstall
   - Or: Program Files\PasteToFile\Uninstall.exe
   - Registry cleaned automatically
